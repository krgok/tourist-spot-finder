観光地調査アプリ

API設定

観光地情報・地図・経路表示には Google Maps / Places API / Routes API のAPIキーが必要です。 ご自身の Google Cloud Console で「Maps JavaScript API」「Places API (New)」「Routes API」を有効化したAPIキーを取得し、以下に入力してください（ブラウザの localStorage にのみ保存されます）。 ログインはアプリ運営者が用意したGoogleログイン(Supabase Auth)を利用するため、キーの入力は不要です。

周辺検索

お気に入り

ログインすると、複数端末で共有されるお気に入りを保存できます。

検索履歴

ログインすると、過去の検索履歴が保存されます。
